### （1）制作アプリのタイトル
#### Trio Pulse- inproved-version

### （2）制作アプリの説明（40文字程度）
#### Trio Pulse は、「じゃんけんを演奏に変換するゲーム」です。音ゲーでも、シューティングでもないリズムじゃんけんゲームです。プレイヤーは、迫ってくるノーツを見て、勝てる手、あいこ、負ける手を判断し、拍に合わせて入力します。

### （3）工夫した点・こだわった点
#### ジャンケンアイコンに絵文字を使用していたため使用端末により絵文字表示が変わらないように別のアイコンを実装した。アイコンは一般的なimgではなくSVGを使用することで様々な装飾が可能になるようにした。スマホでサイトを開いた際にタッチ操作に対応できるようにした。

### （4）難しかった点・次回トライしたいこと（または機能）
#### 前回作成した課題にバグが見つかったため可能な限り改善につとめた。具体的にはスマホでの表示の際にアイコン表示が消失する、ブラウザでキー操作が無効になるなどの現象が発覚した。ユーザーの視点に立って機能を追加しようと意気込んだ結果、これまで機能していた操作が意図せず機能しなくなる事態が起きることを学んだ。今回始めてデバッグというものに挑戦してみた。本来ならもう少デザインの改良を行う予定であったが、開発期間までのバグの解消を最優先とし、一部変更を取捨選択する決断をした。今回は以前にもまして開発の時間のがなく、更にバグがなかなか改善しない言う状況に陥り、完全に時間との戦いになった。これを書いているさなかもまだバグは解消して状況だであるが残り時間もないので先にREADMEを書いているという状況です。とにかく締め切り時間いっぱいまで動くように挑戦してみます。
