（1）制作アプリのタイトル
 Trio Pulse
（2）制作アプリの説明（40文字程度）
 Trio Pulse は、「じゃんけんを演奏に変換するゲーム」です。音ゲーでも、シューティングでもないリズムじゃんけんゲームです。プレイヤーは、迫ってくるノーツを見て、勝てる手、あいこ、負ける手を判断し、拍に合わせて入力します。

（3）工夫した点・こだわった点
操作の判定タイミング、ゲーム性などユーザーが一回でもリトライしてみようかと思う仕様を盛り込みました。演出に関してはイメージ画像は一切使用せずに、HTML,CSS,JaveScriptのみで実装できる演出のみで作成する縛りでどこまでできるか挑戦しました。その過程でJavaScriptでも簡単な音が出せることがわかりました。じゃんけんアイコンは普通に文字変換で出てくる:fist:️:v::raised_hand_with_fingers_splayed:の絵文字をそのまま使いました。難易度によってノーツを処理するための条件が複雑になるようにし、飽きが来ないようにしました。（あいこを出す、わざと負けるなど）。難易度が高いほどスコアも高くなります。スマホ使用できるように縦長のレイアウトで作成しました。
（4）難しかった点・次回トライしたいこと（または機能）
 今回はついにAIの使用を解禁しました。（とはいえまだ無課金AI）。基礎的な課題はAIは使わずにわからないことをググって自力でコーディングしました。そのあと別の創作物（今回提出したTrio Pules）のコーディングはAIの補助を全力で活用し、いわゆるAIとどのように共同作業するかを学ぶ実験をしました。指示出しでコーディングにどんな影響が出るか何度もラリーしました。はじめはイカサマを見破るじゃんけんを作る予定でした。試しにAIに相談したらその時点で一瞬で試作品がで出力されて絶望しました。（いや、すごいなと関心しました）
 イカサマじゃんけんはAIが仕様を決め一瞬でできたので、作成過程に自分がワクワクしませんでした。なので改めて自分がワクワクするものを作ろうと思い別途新しい構想でリズムゲーム（Trio Pulse）を作成しました。Trio PulseはAIとラリーを繰り返しながら楽しく作成しました。
 イメージ画像を用いた演出はトライできていないので、今後はイメージ画像の作成についても挑戦しもう少しデザイン性にも配慮していきたい。また、今回は音声ファイルも使用していないので音響効果も機会があれば今後の作品で取り組んでみたい。
（5）備考（感想、シェアしたいこと等なんでも）
 無課金AIでここまでできたので、今後は何かしらの課金AIを使用し、性能がどのように変わるのかを体感してみたい。AIを使うことでこれまで経験した「手を動かす」作業が圧倒的に減りました。それが良いことかどうかはさておき、今回一つのゲームを作成できたことは素直に喜んで手伝ってもらったAIさんに感謝したいです。
